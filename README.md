# 🎮 GameHub App

Aplicativo desenvolvido para exibição e organização de jogos, com foco em interface moderna, reutilização de componentes e boas práticas de desenvolvimento.

O projeto foi criado com objetivo educacional, visando aprimorar habilidades em desenvolvimento mobile utilizando tecnologias modernas.

---

## 📌 Funcionalidades

- Listagem de jogos  
- Interface organizada e responsiva  
- Estrutura baseada em componentes reutilizáveis  
- Navegação entre telas  
- Código modular e escalável  

---

## 🛠️ Tecnologias utilizadas

- React Native  
- Expo  
- TypeScript  
- JavaScript  

---

## 🚀 Como executar o projeto

```bash
git clone https://github.com/rafasantossss/gamehub-app
cd gamehub-app
npm install
npx expo start
